About this site. M.E.A.N Stack Development.

This site was created using MongoDB, Express, AngularJS, and NodeJS, also know as the M.E.A.N Stack. It was deployed using Heroku via GitHub, where the code can be viewed. This site implements many features including user registration with form validation, e-mail varification, temporary tokens to keep users logged in, options to loggin in through third parties like Facebook or Twitter, and User restrictions. It is an example of a small portion of the possiblites that programming has to offer.
